cli-tools [subcommand]

  help                 Displays help.
  install              Installes CLI subcommands.
  update               Upgrades CLI subcommands.
  uninstall            Uninstalles CLI subcommands.
  alias <subcommand>
  rm <subcommand>

  deploy              Updates k8s deployment.
  jell                JavaScript shell.
  latex               Compiles LaTeX document.
  serve               Starts HTTP server.
  update-node-dep     Update all node dependencies.
